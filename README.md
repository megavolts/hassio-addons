## Installation

[!["Add repository on my Home Assistant"](https://raw.githubusercontent.com/Smeagolworms4/donate-assets/master/addon-ha.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2Fmegavolts%2Fhassio-addons)

This repository can be added manually following this procedure [Home Assistant website](https://home-assistant.io/hassio/installing_third_party_addons). The URL to add this repository is : [https://github.com/megavolts/hassio-addons](https://github.com/megavolts/hassio-addons)


### Detailed Instructions

1. Navigate in your Home Assistant frontend to <kbd>Supervisor</kbd> -> <kbd>Add-on Store</kbd>.

2. Click the 3-dots menu at upper right <kbd>...</kbd> > <kbd>Repositories</kbd> and add the repository URl.
